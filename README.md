# Notes Markdown

Una aplicación de notas simple construida con TypeScript y Vite, con soporte para Markdown.

## Características

- Crear, editar y eliminar notas
- Soporte para renderizado de Markdown
- Almacenamiento local en el navegador
- Interfaz simple e intuitiva

## Tecnologías

- TypeScript
- Vite
- MarkdownIt (para renderizado de Markdown)

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Producción

```bash
npm run build
```
